# 🧮 Smart Calculator Web

**A complete, sleek, fully functional web application** replicating the popular Android app “Calculator: Simple Calculator” with Currency Converter, Tip Calculator, Unit Converter and more.

## ✨ Features
- 🧮 **Basic + Scientific Calculator** – Full keyboard with sin/cos/tan/log/√  
- 💱 **Live Currency Converter** – 10+ currencies with instant conversion  
- 💵 **Tip Calculator** – Bill split, custom %, slider & per-person total  
- 📏 **Unit Converter** – Length, Weight, Volume, Area, Time & Data  
- 🏷️ **Extra Tools** – Discount, GST/Tax, Loan & Date Difference  
- 📱 **100% Mobile-First** – Beautiful on phones, tablets & desktop  
- 🌐 **SEO Optimized** – Meta tags, keywords & lightning-fast performance  
- 💾 **Offline Ready** – All calculations work without internet  

## 🛠️ Technologies Used
- 🔨 **HTML5** – Semantic structure  
- 🎨 **CSS3** – Modern dark theme + smooth animations  
- 📱 **Bootstrap 5** – Fully responsive & clean UI  
- ⚡ **JavaScript + jQuery** – Real-time math & conversions  

## 🚀 How to Run
1. Download the 4 files into the **same folder**  
2. Open `index.html` in any browser  
3. Start calculating instantly!
